# ShowEffect 指令

##描述
显示指定特效。
参数一为特效 id，参数二为角色(Character类)。
##参数


｜序号｜ 参数 ｜是否可选｜          描述  ｜
｜:--:｜:----:｜:------:｜:--------:｜
｜1  ｜ 特效 id ｜   否   ｜指定要展示的特效的 id ｜
｜2  ｜ 角色 ｜   是   ｜在指定角色身上显示特效；<br/>为空则在目标上显示特效｜

##示例
###示例一：
#### excel-Card 表格

｜参考列｜    表头    ｜ 值 ｜
｜:----:｜:----------:｜:--:｜
｜  A   ｜     id     ｜  150(请根据实际情况填写) ｜
｜  B   ｜displayName ｜  测试卡牌  ｜
｜…略…｜            ｜    ｜
｜  M   ｜    code    ｜  ShowEffect:6 ｜
｜…略…｜            ｜    ｜

**结果：在目标身上显示 id 为 6 的特效**
###示例二：
#### excel-Card 表格

｜参考列｜    表头    ｜ 值 ｜
｜:----:｜:----------:｜:--:｜
｜  A   ｜     id     ｜  150(请根据实际情况填写) ｜
｜  B   ｜displayName ｜  测试卡牌  ｜
｜…略…｜            ｜    ｜
｜  M   ｜    code    ｜  ShowEffect:6,{get_user:} ｜
｜…略…｜            ｜    ｜

**结果：在使用者身上显示 id 为 6 的特效**
