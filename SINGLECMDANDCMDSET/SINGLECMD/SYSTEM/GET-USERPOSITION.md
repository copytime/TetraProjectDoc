# get_userPosition 指令

##描述
获取玩家的位置。
返回值为三维向量。
##参数

<1.2>无参数

##示例
#### excel-Card 表格

｜参考列｜    表头    ｜ 值 ｜
｜:----:｜:----------:｜:--:｜
｜  A   ｜     id     ｜  150(请根据实际情况填写) ｜
｜  B   ｜displayName ｜  测试卡牌  ｜
｜…略…｜            ｜    ｜
｜  M   ｜    code    ｜  CreateEnemy:1,{get_userPosition:} ｜
｜…略…｜            ｜    ｜

**结果：在玩家所在位置生成角色 id 为 1 的敌人，此时会生成在玩家身边的格子上**
